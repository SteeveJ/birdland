## Chargement du menu et du footer
===============================
Ajouter au début de votre body ou de votre container la ligne suivante pour le menu : 

```html
    <div id="navbar"></div>
```

Pour le footer à la fin de votre body ou container :

```html
    <div id="footer"></div>
```
